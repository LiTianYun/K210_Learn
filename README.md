# K210_Learn
学习K210的一些例程与资料
=====================
### 硬件基于歪朵拉K210模块 维基主页：https://widora.io/k210
### 官方IDE：http://kendryte-ide.s3-website.cn-northwest-1.amazonaws.com.cn/alpha.html

## 目录：

`examples`            存储测试完成可直接使用的例程<br>
`modle`                   存储可使用模型<br>
`modle_test`         存储测试中模型<br>
`docs`                       存储相关文档<br>
`workspace`           工作空间 存储正在编写的程序<br>
